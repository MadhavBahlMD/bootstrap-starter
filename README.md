## Include in the box

- Scss compiler
- Bootstrap (4.0.0-beta-2) and Font Awesome (4.7.0)
- Bourbon (4.3.4)

## Installation
To install simply just clone with `git clone`

then `npm install` or `yarn install`

then `npm run dev` or `yarn run dev`
